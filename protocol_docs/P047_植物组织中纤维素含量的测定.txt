四、所需仪器与材料
( 一 ) 材 料
烘干的米、面粉或风干的棉、麻纤维。
(二)仪器设备
分析天平，水浴锅，电炉，小试管，量筒，烧杯，移液管，容量瓶，布氏漏斗，分 光光度计。
( 三 ) 试 剂
(1)60% H₂SO₄ 溶 液 。
( 2 ) 浓 H₂SO₄(AR )。

五、所需试剂
沿管壁加5 mL 浓 H₂SO₄,塞上塞子，摇匀，静置12 min, 然后在620 nm 波长下测
吸光度。
