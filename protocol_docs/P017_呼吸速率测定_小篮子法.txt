Protocol编号：P017
实验名称：呼吸速率测定 - 小篮子法/广口瓶法（李合生版）

一、一句话说清
植物释放的CO2被Ba(OH)2吸收，用草酸滴定剩余碱液，算出CO2释放量就是呼吸速率。

二、实验原理
植物呼吸作用释放CO2，用定量的Ba(OH)2溶液吸收。实验结束后用已知浓度的草酸溶液滴定剩余的碱液，从空白和样品二者消耗草酸溶液之差计算呼吸过程中释放的CO2量。
每毫升1/44 mol/L草酸相当于1mg CO2。

三、来源出处
- 李合生主编《植物生理生化实验原理和技术》，高等教育出版社
- 实验21，小篮子法（广口瓶法）测定植物的呼吸速率

四、所需仪器
1. 500mL广口瓶
2. 三孔橡皮塞
3. 碱石灰干燥管
4. 温度计
5. 酸式滴定管
6. 小篮子
7. 天平
8. 计时器

五、所需试剂
1. 0.05mol/L Ba(OH)2溶液
2. 1/44mol/L草酸溶液
3. 酚酞指示剂
4. 碱石灰

六、试剂配方
1. 0.05mol/L Ba(OH)2：
 称取Ba(OH)2 8.6g溶于1000mL蒸馏水中

2. 1/44mol/L草酸溶液：
 准确称取重结晶H2C2O4·2H2O 2.8651g
 溶于蒸馏水定容至1000mL
 每毫升相当于1mg CO2

3. 酚酞指示剂：
 1g酚酞溶于100mL 95%乙醇中

七、分步操作步骤
步骤1：装置准备
 取500mL广口瓶，用三孔橡皮塞塞紧
 一孔插碱石灰干燥管（进入无CO2空气）
 一孔插温度计
 一孔供滴定用（平时用小橡皮塞塞紧）
 瓶塞下面挂小篮子
 为什么：碱石灰除去空气中CO2，避免干扰
 做对了：装置密封良好
 做错了：漏气导致外界CO2进入

步骤2：空白测定
 在瓶中准确加入20mL Ba(OH)2溶液
 塞紧橡皮塞（不带小篮子）
 充分摇动广口瓶2min
 待瓶内CO2全部被吸收后
 加入2滴酚酞指示剂
 用草酸滴定至无色
 记录空白消耗草酸量A(mL)
 为什么：空白排除装置中残留CO2的干扰
 做对了：粉红色消失为无色
 做错了：滴过终点

步骤3：样品测定
 倒出废液，用无CO2蒸馏水洗净
 重新加入20mL Ba(OH)2溶液
 称取植物样品5~10g装入小篮子
 挂于橡皮塞下，塞紧瓶塞
 记录开始时间
 准确计时30min~1h
 迅速取出小篮子
 加入2滴酚酞指示剂
 立即塞紧塞子，充分摇动2min
 用草酸滴定至无色
 记录样品消耗草酸量B(mL)
 为什么：Ba(OH)2吸收呼吸释放的CO2
 做对了：样品消耗草酸比空白少
 做错了：口中呼出气体进入瓶内

八、安全红线
1. Ba(OH)2有腐蚀性，戴手套操作
2. 草酸有腐蚀性，避免接触皮肤
3. 防止口中呼出气体进入瓶内

九、新手避坑提示
1. 蒸馏水需煮沸冷却（无CO2）后使用
2. 防止口中呼出气体进入瓶内
3. 滴定速度要慢，接近终点时逐滴加入
4. 计时要准确
5. 每个样品做3个重复

十、数据处理与结果判读
计算公式：
呼吸速率(mg CO2/g FW/h) = (A - B) × 1 / (W × t)
A = 空白滴定用去的草酸量(mL)
B = 样品滴定用去的草酸量(mL)
W = 样品鲜重(g)
t = 测定时间(h)
每毫升1/44mol/L草酸 = 1mg CO2

十一、常见问题排查表
问题：空白和样品差异小 → 原因：呼吸速率低 → 解决：增加样品量或延长测定时间
问题：数据波动大 → 原因：装置漏气 → 解决：检查密封性
问题：滴定终点不明显 → 原因：Ba(OH)2浓度不对 → 解决：重新配制
