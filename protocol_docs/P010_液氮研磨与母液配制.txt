Protocol编号：P010
实验名称：液氮研磨与试剂母液配制（基础操作）

一、一句话说清
液氮研磨就是把植物组织冻到-196°C变脆，然后用研钵捣碎；母液就是高浓度的试剂原液，用的时候稀释就行。

二、实验原理
液氮温度-196°C，能使含水的植物组织瞬间冻结变脆，研磨时容易破碎成细粉。同时低温灭活酶类（蛋白酶、RNase等），保护目标分子不被降解。母液(MStock Solution)是浓缩的试剂储备液，使用时按比例稀释为工作液，避免每次都称量微量试剂。

三、来源出处
- 基础实验操作，参考《分子克隆实验指南》(Sambrook & Russell)
- 《植物生理学实验指导》基础操作部分

四、液氮研磨操作

所需物品：
1. 液氮罐及液氮
2. 研钵和研棒（提前清洗晾干）
3. 不锈钢勺或药匙
4. 防冻手套（必须！）
5. 护目镜（建议）
6. 1.5mL离心管

分步操作：
步骤1：戴好防冻手套和护目镜
 为什么：液氮-196°C，裸手接触会严重冻伤
 安全红线：永远不要裸手碰液氮或刚接触液氮的物品！

步骤2：从液氮罐中取出适量液氮倒入研钵（约1/3满）
 为什么：预冷研钵
 做对了：液氮在研钵中安静沸腾
 做错了：研钵是湿的→液氮飞溅

步骤3：将植物组织（约0.1-0.2g）放入研钵
 为什么：组织在液氮中进一步冷冻
 做对了：组织变硬变脆

步骤4：等液氮快要蒸发完时开始研磨
 为什么：液氮太多会溅出来，太少组织会回温
 做对了：听到"沙沙"声，粉末细腻
 做错了：液氮没蒸发就研磨→液氮飞溅；等太久→组织回温变软

步骤5：持续研磨至粉末状，补充液氮保持低温
 做对了：粉末均匀细腻，无明显颗粒
 做错了：研磨不充分→后续提取效率低

步骤6：用预冷的勺子将粉末转移到预冷的离心管中
 做对了：粉末完整转移
 做错了：管子不预冷→粉末回温

步骤7：立即加入提取液（如TRIzol、CTAB提取液等）
 为什么：防止样品回温后酶复活降解目标分子
 做对了：加液后立即混匀
 做错了：放着不加液→酶开始降解

避坑提示：
1. 研钵必须干燥！有水的话液氮会剧烈飞溅
2. 研磨时研棒不要抬太高，防止粉末飞出
3. 液氮用完的研钵不要马上用水洗，等回到室温再洗
4. 液氮罐存放要通风，密闭空间液氮蒸发会造成窒息

五、试剂母液配制操作

母液配制通用公式：
称取质量(g) = 摩尔浓度(mol/L) × 分子量(g/mol) × 体积(L)

常见母液配制清单：

1. 1mol/L Tris-HCl(pH8.0) 100mL
 称取12.11g Tris → 溶于80mL蒸馏水
 用浓盐酸调pH至8.0（约加4.2mL浓HCl）
 定容至100mL，高压灭菌

2. 0.5mol/L EDTA(pH8.0) 100mL
 称取18.61g EDTA-2Na → 溶于80mL蒸馏水
 用NaOH调pH至8.0（EDTA在pH<8.0时不溶！）
 定容至100mL，高压灭菌

3. 5mol/L NaCl 100mL
 称取29.22g NaCl → 蒸馏水定容至100mL
 高压灭菌

4. 1mol/L CaCl₂ 100mL
 称取11.1g CaCl₂ → 蒸馏水定容至100mL
 高压灭菌

5. 1mol/L MgCl₂ 100mL
 称取9.52g MgCl₂ → 蒸馏水定容至100mL
 高压灭菌

梯度稀释操作（标准曲线常用）：
方法：从高浓度到低浓度逐级稀释
例：从1mg/mL母液做2倍系列稀释
 第0管：取500μL母液（1mg/mL）
 第1管：取250μL母液 + 250μL溶剂 = 0.5mg/mL
 第2管：取250μL第1管液 + 250μL溶剂 = 0.25mg/mL
 第3管：取250μL第2管液 + 250μL溶剂 = 0.125mg/mL
 以此类推

避坑提示：
1. 母液标签必须写：名称、浓度、配制日期、配制人
2. Tris的pH要用盐酸调，不能用硫酸（硫酸盐会沉淀）
3. EDTA必须用NaOH调到pH8.0以上才溶解，别以为是试剂坏了
4. 母液4°C保存一般可用3-6个月，出现浑浊/沉淀/长菌就扔掉
5. 梯度稀释时每一步都要换枪头，避免交叉污染
6. 稀释时先加溶剂再加母液，混匀后取下一步的量

六、安全红线
1. 液氮-196°C，永远不要裸手接触液氮或刚接触液氮的物品！必须戴防冻手套
2. 液氮罐存放必须通风，密闭空间液氮蒸发会造成窒息
3. 液氮用完的研钵不要马上用水洗，等回到室温再洗，否则温差过大会炸裂
4. 浓盐酸、NaOH有强腐蚀性，调pH时戴手套护目镜

八、数据处理
母液配制称量计算：
称取质量(g) = 摩尔浓度(mol/L) × 分子量(g/mol) × 体积(L)

例：配100mL 0.5mol/L EDTA(pH8.0)
EDTA-2Na分子量=372.24
称取质量 = 0.5 × 372.24 × 0.1 = 18.61g

梯度稀释浓度计算：
第n管浓度 = 母液浓度 × (1/2)ⁿ
例：1mg/mL母液做2倍系列稀释
 第0管：1mg/mL
 第1管：0.5mg/mL
 第2管：0.25mg/mL
 第3管：0.125mg/mL

九、常见问题排查表
问题：液氮飞溅 → 原因：研钵有水或液氮加太满 → 解决：确保研钵干燥，液氮加至1/3满
问题：研磨时组织变软 → 原因：液氮蒸发完后组织回温 → 解决：及时补充液氮，保持低温
问题：EDTA不溶解 → 原因：pH未调至8.0以上 → 解决：用NaOH调pH至8.0，EDTA在酸性条件下不溶
问题：Tris母液pH偏移 → 原因：Tris的pH受温度影响大 → 解决：在室温下校准pH，标注配制温度

十、rpm与rcf换算说明
很多protocol写的离心条件用rcf(×g)，但离心机面板上显示的是rpm，需要换算：
rcf = 1.118 × 10⁻⁵ × r × rpm²
r = 转子半径(mm)，看转子上的标注

常见转子参考：
- 水平转子(如Eppendorf A-4-44)：r=157mm
 4000rpm = 2812×g
 3000rpm = 1582×g
- 角转子(如Eppendorf F-45-30-11)：r=85mm
 12000rpm = 13734×g
 10000rpm = 9515×g

也可以用植研小白盒的试剂计算器直接换算。
