Protocol编号：P008
实验名称：琼脂糖凝胶电泳

一、一句话说清
把DNA样品加到琼脂糖凝胶的小孔里，通电后DNA按大小分开，大片段跑得慢在上面，小片段跑得快在下面。

二、实验原理
DNA带负电荷，在电场中向正极迁移。琼脂糖凝胶形成网状结构，小DNA片段穿过网孔快，大片段慢，从而实现按大小分离。用核酸染料(如GelRed)染色后在紫外灯下可见DNA条带。

三、来源出处
- Sambrook, J. & Russell, D.W. Molecular Cloning: A Laboratory Manual
- 标准分子生物学实验方法

四、所需仪器
1. 电泳槽（水平式）
2. 电泳仪（能输出100-150V直流电）
3. 微波炉或电炉
4. 凝胶成像系统（或紫外透射仪）
5. 电子天平
6. 移液枪

五、所需试剂
1. 琼脂糖 AR
2. 1×TAE缓冲液（或1×TBE缓冲液）
3. 核酸染料 GelRed（替代有毒的EB溴化乙锭）
4. 6×上样缓冲液（含甘油和溴酚蓝）
5. DNA Marker（已知大小的DNA片段混合物）
6. 蒸馏水

六、试剂配方
1. 50×TAE缓冲液（母液）：
 Tris 242g + 冰醋酸57.1mL + 0.5mol/L EDTA(pH8.0) 100mL
 定容至1000mL，使用时稀释50倍为1×TAE

2. 1×TAE工作液：
 取50×TAE 20mL + 蒸馏水980mL

3. 1%琼脂糖凝胶：
 称0.5g琼脂糖 + 50mL 1×TAE
 微波炉加热溶解（中火2-3min，中途取出摇匀）
 冷却至60°C加2.5μL GelRed，混匀倒胶

4. 6×上样缓冲液：
 甘油30mL + 溴酚蓝0.25g + 二甲苯青FF 0.25g + 蒸馏水定容至100mL

七、分步操作步骤
步骤1：配制1%琼脂糖凝胶
 为什么：1%适合分离200bp-10kb的DNA片段
 做对了：琼脂糖完全溶解，溶液透明
 做错了：没溶解完导致凝胶不均匀

步骤2：冷却至60°C，加GelRed核酸染料，混匀倒胶
 为什么：高温会破坏GelRed
 做对了：染料均匀分布
 做错了：太热加染料导致染料失效

步骤3：插入梳子，等待凝固（约20-30min）
 为什么：梳子形成加样孔
 做对了：凝胶完全凝固，拔梳子时孔完整
 做错了：没凝固就拔梳子→孔变形

步骤4：将凝胶放入电泳槽，加1×TAE缓冲液没过凝胶约2mm
 为什么：缓冲液导电，提供电场环境
 做对了：缓冲液刚好没过凝胶表面
 做错了：缓冲液太少→凝胶局部干燥→条带扭曲

步骤5：DNA样品 + 6×上样缓冲液混匀（5:1体积比），加入加样孔
 为什么：上样缓冲液增加密度使样品沉入孔底，溴酚蓝指示电泳进度
 做对了：蓝色样品沉入孔底
 做错了：枪头戳破凝胶孔底→样品漏出

步骤6：加DNA Marker到旁边孔
 为什么：Marker用于判断样品DNA大小
 做对了：Marker和样品在同一块胶上

步骤7：盖上盖子，接通电源，100-120V电泳30-40min
 为什么：电压决定迁移速度
 做对了：蓝色染料迁移至凝胶2/3处
 做错了：电压太高→凝胶发热融化；方向接反→DNA反向迁移

步骤8：关闭电源，取出凝胶，在凝胶成像系统下拍照
 为什么：紫外灯激发GelRed荧光，看到DNA条带
 做对了：看到清晰条带
 做错了：忘记拍照→条带降解后就看不到了

八、安全红线
1. GelRed比EB安全但仍要戴手套操作
2. 微波炉加热琼脂糖时注意暴沸，分次加热
3. 紫外灯下观察必须戴护目镜，不能直视紫外光
4. 电泳通电时不能碰缓冲液

九、新手避坑提示
1. 倒胶时不能有气泡，气泡会导致条带断裂
2. 拔梳子要垂直向上拔，歪了会破坏加样孔
3. 加样时枪头伸入孔中但不要戳到底，轻轻推出样品
4. 正负极不能接反！红色是正极，DNA往红色方向跑
5. 溴酚蓝跑到凝胶2/3处就可以停了，跑太远小片段会跑出凝胶

十、结果判读
- 成功：在目标大小位置有明亮条带，与Marker对比确认大小
- 条带模糊：DNA降解或电压太高
- 拖尾：DNA降解或上样量太多
- 无条带：PCR无产物或上样量太少

十一、常见问题排查表
问题：无条带 → 原因：PCR产物少/染料失效/曝光不足 → 解决：增加上样量，新鲜配GelRed
问题：条带模糊 → 原因：DNA降解/电压太高 → 解决：检查DNA质量，降低电压
问题：孔周围有亮光 → 原因：蛋白或多糖污染 → 解决：重新提取DNA
问题：条带弯曲 → 原因：凝胶不均匀或电泳槽不平 → 解决：重新倒胶
