Protocol编号：P002
实验名称：考马斯亮蓝法（Bradford法）测可溶性蛋白含量

一、一句话说清
考马斯亮蓝染料和蛋白质结合后变蓝色，蛋白越多颜色越深，测595nm吸光度就能算蛋白含量。

二、实验原理
考马斯亮蓝G-250在酸性条件下与蛋白质结合，染料的最大吸收峰从465nm转移到595nm，吸光度与蛋白质含量在一定范围内成正比。用牛血清白蛋白（BSA）做标准曲线定量。

三、来源出处
- Bradford, M.M. (1976) Analytical Biochemistry, 72(1-2): 248-254. DOI: 10.1016/0003-2697(76)90527-3
- 高继国《植物生理学实验指导》实验14

四、所需仪器
1. 紫外分光光度计（能测595nm）
2. 电子天平（精度0.001g）
3. 离心机（最好冷冻，12000rpm）
4. 移液枪（100μL/1mL/5mL）
5. 研钵、液氮罐
6. 比色皿（玻璃或石英）

五、所需试剂
1. 考马斯亮蓝G-250 AR
2. 牛血清白蛋白(BSA) 标准品
3. 95%乙醇 AR
4. 85%磷酸 AR
5. PBS缓冲液(pH7.0) 或蒸馏水

六、试剂配方
1. Bradford试剂：
 称取100mg考马斯亮蓝G-250 → 溶于50mL 95%乙醇
 加入100mL 85%磷酸 → 蒸馏水定容至1000mL
 过滤或离心去除不溶颗粒 → 棕色瓶4°C保存
 保质期1个月

2. BSA标准液（母液0.1mg/mL）：
 称取10mg BSA → 蒸馏水定容至100mL
 4°C保存，可用2周

七、分步操作步骤
步骤1：称取0.2g叶片，液氮研磨成粉末
 为什么：液氮使细胞破碎，释放胞内蛋白
 做对了：粉末细腻均匀
 做错了：研磨不充分导致蛋白提取不完全

步骤2：加5mL PBS缓冲液(pH7.0)或蒸馏水，冰浴中研磨匀浆
 为什么：缓冲液保持pH稳定，冰浴防止蛋白降解
 做对了：匀浆均匀无大块
 做错了：室温操作导致蛋白酶降解蛋白，数据偏低

步骤3：12000rpm 4°C离心20min，取上清
 为什么：高速冷冻离心去除细胞碎片，保持蛋白活性
 做对了：上清澄清
 做错了：不冷冻导致蛋白变性；转速不够导致上清浑浊

步骤4：取0.1mL上清 + 5mL Bradford试剂，混匀
 为什么：Bradford试剂与蛋白反应显色
 做对了：加入后立即混匀，溶液变为蓝色
 做错了：没混匀导致颜色不均匀，读数不准

步骤5：室温静置5min，595nm测吸光度
 为什么：5min后颜色稳定可测量
 做对了：OD值在0.1-0.8之间
 做错了：放置太久(>2h)颜色会沉淀，读数下降

八、安全红线
1. 考马斯亮蓝会染色皮肤和衣服，必须戴手套操作
2. 磷酸有腐蚀性，配试剂时戴护目镜
3. 比色皿用后立即清洗，残留染料影响下次测量

九、新手避坑提示
1. Bradford试剂使用前必须过滤或离心，未过滤的染料颗粒会导致读数不准
2. 比色皿用后立即用乙醇清洗，考马斯亮蓝残留很难洗掉
3. 标准曲线线性范围0-1.5mg/mL，样品OD超出范围需稀释后重测
4. 蛋白样品提取后尽快测量，4°C保存不超过24h
5. 不同蛋白与染料结合能力不同，结果以BSA当量表示

十、数据处理与结果判读
计算公式：
可溶性蛋白含量(mg/g FW) = (C × V × D) / (W × 1000)
C = 从标准曲线查得的蛋白浓度(mg/mL)
V = 提取液总体积(mL)，本实验为5mL
D = 稀释倍数
W = 样品鲜重(g)

十一、常见问题排查表
问题：标准曲线不成线性 → 原因：Bradford试剂有沉淀或BSA变质 → 解决：过滤Bradford试剂，重新配BSA
问题：样品颜色太深 → 原因：蛋白浓度太高 → 解决：稀释样品5-10倍后重测
问题：重复间差异大 → 原因：移液不准确或混匀不充分 → 解决：校准移液枪，加样后立即颠倒混匀
问题：空白读数高 → 原因：比色皿有残留或试剂污染 → 解决：清洗比色皿，更换试剂
