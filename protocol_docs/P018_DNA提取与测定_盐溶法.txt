Protocol编号：P018
实验名称：植物组织DNA提取与测定 - 盐溶法（李合生版）

一、一句话说清
用SDS裂解细胞释放DNA，氯仿-异戊醇去蛋白，乙醇沉淀DNA，二苯胺法测含量。

二、实验原理
盐溶法是提取DNA的常规技术。细胞破碎时DNase会降解DNA，提取缓冲液中加入柠檬酸盐和EDTA可抑制DNase活性并使蛋白质变性。加入SDS使蛋白质与核酸分离，用氯仿-异戊醇去除蛋白质，用乙醇沉淀DNA。
DNA含量测定：DNA被酸水解后产生的脱氧核糖与二苯胺乙醛试剂反应生成蓝色化合物，595nm有最大吸收。

三、来源出处
- 李合生主编《植物生理生化实验原理和技术》，高等教育出版社
- 实验37，植物组织中DNA的提取与测定

四、所需仪器
1. 研钵
2. 离心机
3. 恒温水浴锅（72°C和37°C）
4. 紫外分光光度计
5. 具塞刻度试管
6. 三角瓶
7. 离心管

五、所需试剂
1. NaCl、柠檬酸三钠、EDTA-Na
2. SDS（十二烷基硫酸钠）
3. 氯仿-异戊醇（24:1）
4. 95%乙醇
5. RNase
6. 高氯酸钠
7. 二苯胺、冰醋酸、浓硫酸、乙醛
8. DNA标准品

六、试剂配方
1. 研磨缓冲液：
 NaCl 59.63g + 柠檬酸三钠14.71g + EDTA-Na 37.2g
 分别溶解后合并，用0.2mol/L NaOH调至pH7.0
 定容至1000mL

2. 10×SSC溶液：
 NaCl 87.66g + 柠檬酸三钠44.12g
 定容至1000mL

3. 1×SSC和0.1×SSC：分别用10×SSC稀释10倍和100倍

4. RNase溶液：
 用0.14mol/L NaCl配制25mg/mL
 用1mol/L HCl调pH至5.0
 使用前80°C水浴处理5min（破坏DNase）

5. 氯仿-异戊醇：
 氯仿24mL + 异戊醇1mL混合

6. 5mol/L高氯酸钠：
 NaClO4·H2O 70.23g，加少量蒸馏水溶解，定容至100mL

7. 二苯胺乙醛试剂：
 1.5g二苯胺溶于100mL冰醋酸
 添加1.5mL浓硫酸
 棕色瓶暗处保存
 使用时加0.1mL乙醛液（浓乙醛:水=1:50）

8. DNA标准液：
 标准DNA 25mg溶于少量0.05mol/L NaOH
 定容至25mL，吸取5mL至50mL容量瓶
 加5.0mL 1mol/L HClO4，冷却后用0.5mol/L HClO4定容

七、分步操作步骤
步骤1：DNA提取
 称取去胚乳小麦芽10g（或幼嫩组织），剪碎
 加10mL预冷研磨缓冲液 + 0.1g SDS
 置冰浴上研磨成糊状
 为什么：SDS裂解细胞，EDTA抑制DNase
 做对了：匀浆均匀
 做错了：研磨不充分导致DNA释放不完全

步骤2：去蛋白
 将匀浆转入25mL刻度试管
 加入等体积氯仿-异戊醇混合液
 加塞剧烈振荡0.5min
 4000r/min离心5min
 小心吸取上层清液（含DNA）
 为什么：氯仿-异戊醇去除蛋白质
 做对了：分为三层，上层含DNA
 做错了：吸到中间层导致蛋白污染

步骤3：灭活DNase
 试管置72°C水浴保温3min（不超过4min）
 迅速取出置冰水浴冷却至室温
 加5mol/L高氯酸钠（体积比4:1）
 使高氯酸钠最终浓度为1mol/L
 为什么：72°C灭活DNase，高氯酸钠助溶
 做对了：溶液澄清
 做错了：温度过高或时间过长导致DNA降解

步骤4：再次去蛋白
 再加入等体积氯仿-异戊醇
 振荡1min，静置
 4000r/min离心5min
 取上清液置小烧杯

步骤5：乙醇沉淀DNA
 用滴管吸95%预冷乙醇
 慢慢加入上清液表面
 直至乙醇体积为上清液的两倍
 用玻璃棒轻轻搅动
 核酸以纤维状沉淀缠绕在玻璃棒上
 为什么：乙醇降低溶解度使DNA沉淀
 做对了：白色纤维状沉淀
 做错了：乙醇加入太快导致DNA断裂

步骤6：溶解DNA
 将沉淀在烧杯壁上轻轻挤压除去乙醇
 用5mL 0.1×SSC溶解
 加入0.5mL 10×SSC，使最终为1×SSC
 重复步骤5和6得到DNA粗制品

步骤7：去除RNA（纯化可选）
 加入RNase溶液（终浓度50~70μg/mL）
 37°C水浴保温30min
 加等体积氯仿-异戊醇振荡1min
 4000r/min离心5min
 取上层水溶液
 再次乙醇沉淀得到纯化DNA

步骤8：DNA鉴定（紫外吸收法）
 在260nm和280nm测吸光度
 A260/A280 = 1.8表示纯度好
 A260/A280 < 1.8表示有蛋白污染

步骤9：DNA含量测定（二苯胺法）
 取2mL DNA提取液 + 4mL二苯胺乙醛试剂
 60°C恒温水浴保温1h
 冷却后595nm测吸光度
 查标准曲线求DNA含量

八、安全红线
1. 氯仿有肝毒性和致癌性，必须在通风柜中操作
2. SDS有刺激性，戴手套和口罩
3. 浓硫酸、高氯酸有强腐蚀性，戴护目镜

九、新手避坑提示
1. 全程低温操作（冰浴），防止DNase降解DNA
2. 72°C灭活DNase不超过4min，否则DNA也会降解
3. 乙醇沉淀时慢慢加入，不要搅动太剧烈
4. A260/A280比值1.8~2.0为纯DNA
5. 幼嫩组织DNA含量高，老组织多糖多不好提取

十、数据处理与结果判读
紫外吸收法：
DNA浓度(μg/mL) = A260 × 50 × 稀释倍数
（1 OD260 ≈ 50μg/mL双链DNA）

二苯胺法：
查标准曲线得DNA含量(μg)
DNA含量(mg/g FW) = (C × V) / (W × 1000)

纯度判断：
A260/A280 = 1.8 → 纯DNA
A260/A280 < 1.8 → 蛋白污染
A260/A280 > 2.0 → RNA污染

十一、常见问题排查表
问题：DNA得率低 → 原因：DNase降解或研磨不充分 → 解决：全程低温，充分研磨
问题：A260/A280偏低 → 原因：蛋白污染 → 解决：增加氯仿-异戊醇抽提次数
问题：DNA断片多 → 原因：操作剧烈或温度过高 → 解决：轻柔操作，控温72°C不超过4min
