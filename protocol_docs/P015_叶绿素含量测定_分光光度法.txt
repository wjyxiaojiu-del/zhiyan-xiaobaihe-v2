Protocol编号：P015
实验名称：叶绿素含量测定 - 分光光度法（李合生版）

一、一句话说清
用乙醇提取叶绿素，测665nm和649nm吸光度，用公式算叶绿素a、b和类胡萝卜素含量。

二、实验原理
根据叶绿体色素提取液对可见光谱的吸收，利用分光光度计在特定波长下测定吸光度，根据朗伯-比尔定律计算色素含量。
叶绿素a、b在95%乙醇中最大吸收峰分别为665nm和649nm，类胡萝卜素为470nm。
吸光度具有加和性：混合液在某一波长下的总吸光度等于各组分在相应波长下吸光度的总和。

三、来源出处
- 李合生主编《植物生理生化实验原理和技术》，高等教育出版社
- 实验12，叶绿素含量的测定

四、所需仪器
1. 分光光度计（能测665/649/470nm）
2. 电子天平（感量0.01g）
3. 研钵
4. 棕色容量瓶（25mL）
5. 小漏斗、定量滤纸
6. 吸水纸、擦镜纸、滴管

五、所需试剂
1. 95%乙醇（或80%丙酮）
2. 石英砂
3. 碳酸钙粉

六、试剂配方
1. 95%乙醇：直接使用分析纯95%乙醇
2. 80%丙酮：取80mL丙酮加蒸馏水定容至100mL（用于干材料）

七、分步操作步骤
步骤1：样品准备
 取新鲜植物叶片，擦净表面污物，剪碎（去掉中脉），混匀
 称取0.2g，共3份
 为什么：去掉中脉避免非光合组织干扰
 做对了：叶片剪成小碎片
 做错了：中脉未去掉导致数据偏低

步骤2：提取
 加少量石英砂和碳酸钙粉及2~3mL 95%乙醇
 研成匀浆，再加乙醇10mL，继续研磨至组织变白
 静置3~5min
 为什么：碳酸钙防止叶绿素被酸破坏，研磨至组织变白说明色素提取完全
 做对了：组织变白，溶液呈绿色
 做错了：研磨不充分导致提取不完全

步骤3：过滤
 取滤纸置漏斗中，用乙醇湿润
 把提取液倒入漏斗中，过滤到25mL棕色容量瓶
 用少量乙醇冲洗研钵、研棒及残渣数次
 用滴管吸取乙醇，将滤纸上色素全部洗入容量瓶
 直至滤纸和残渣中无绿色为止
 用乙醇定容至25mL，摇匀
 为什么：棕色瓶防止光照分解叶绿素
 做对了：残渣无绿色，溶液定容准确
 做错了：残渣有绿色说明提取不完全

步骤4：测定
 把提取液倒入1cm比色杯
 以95%乙醇为空白
 在665nm、649nm和470nm下测定吸光度
 为什么：三个波长分别对应叶绿素a、b和类胡萝卜素
 做对了：三个波长都测了
 做错了：只测一个波长无法计算各组分含量

八、安全红线
1. 乙醇易燃，远离火源
2. 丙酮有毒，通风柜中操作
3. 操作在弱光下进行，防止叶绿素光分解

九、新手避坑提示
1. 操作应在弱光下进行，研磨时间尽量短
2. 提取液不能混浊，可在710nm检测，吸光度应<吸收峰处的5%
3. 分光光度计波长精度要求高，使用前需校正
4. 加碳酸钙防止叶绿素被酸破坏
5. 干材料必须用80%丙酮，新鲜材料可用95%乙醇

十、数据处理与结果判读
计算公式（95%乙醇提取）：
Ca = 13.95A665 - 6.88A649 (mg/L)
Cb = 24.96A649 - 7.32A665 (mg/L)
Ca+b = Ca + Cb
Cx·c = (1000A470 - 2.05Ca - 114.8Cb) / 245 (mg/L)

色素含量(mg/g FW) = (C × V) / (W × 1000)
C = 色素浓度(mg/L)
V = 提取液总体积(mL) = 25mL
W = 样品鲜重(g)

十一、常见问题排查表
问题：提取液浑浊 → 原因：研磨不充分或过滤不彻底 → 解决：重新研磨过滤
问题：数据偏低 → 原因：光照分解或未加CaCO3 → 解决：弱光操作，加CaCO3
问题：重复差异大 → 原因：取样不均匀 → 解决：剪碎混匀后称样
