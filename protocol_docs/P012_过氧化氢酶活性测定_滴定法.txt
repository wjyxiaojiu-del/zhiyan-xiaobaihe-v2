Protocol编号：P012
实验名称：过氧化氢酶(CAT)活性测定 - 高锰酸钾滴定法（李合生版）

一、一句话说清
CAT分解H2O2，用高锰酸钾滴定剩余的H2O2，就能算出CAT分解了多少H2O2。

二、实验原理
过氧化氢酶(CAT)属于血红蛋白酶，含有铁，能催化过氧化氢分解为水和分子氧。在反应系统中加入一定量(过量)的H2O2，经酶促反应后，用标准高锰酸钾溶液(酸性条件下)滴定多余的H2O2，即可求出消耗的H2O2量。
5H2O2 + 2KMnO4 + 4H2SO4 → 5O2 + 2KHSO4 + 8H2O + 2MnSO4

三、来源出处
- 李合生主编《植物生理生化实验原理和技术》，高等教育出版社
- 实验23，过氧化氢酶活性的测定(高锰酸钾滴定法)

四、所需仪器
1. 研钵
2. 三角瓶（50mL）×4
3. 酸式滴定管
4. 恒温水浴（30°C）
5. 容量瓶（25mL）
6. 离心机

五、所需试剂
1. 10% H2SO4
2. 0.2mol/L pH7.8磷酸缓冲液
3. 0.1mol/L高锰酸钾标准液
4. 0.1mol/L H2O2
5. 0.1mol/L草酸

六、试剂配方
1. 0.1mol/L KMnO4标准液：
 称取KMnO4(AR) 3.1605g → 用新煮沸冷却蒸馏水配制1000mL
 再用0.1mol/L草酸溶液标定

2. 0.1mol/L H2O2：
 取30% H2O2 5.68mL → 稀释至1000mL
 用标准0.1mol/L KMnO4溶液(酸性条件下)标定

3. 0.1mol/L草酸：
 称取H2C2O4·2H2O 12.607g → 蒸馏水溶解定容至1000mL

七、分步操作步骤
步骤1：酶液提取
 取小麦叶片2.5g + pH7.8磷酸缓冲液少量，研磨成匀浆
 转移至25mL容量瓶，用缓冲液冲洗研钵并转入容量瓶
 用同一缓冲液定容
 4000r/min离心15min，上清液即为CAT粗提液
 为什么：缓冲液保持pH7.8，维持酶活性
 做对了：上清液澄清
 做错了：pH不对导致酶失活

步骤2：酶促反应
 取50mL三角瓶4个（2个测定+2个对照）
 测定瓶：加入酶液2.5mL
 对照瓶：加入煮死酶液2.5mL
 再加入2.5mL 0.1mol/L H2O2，同时计时
 30°C恒温水浴保温10min
 立即加入10% H2SO4 2.5mL
 为什么：H2SO4终止酶反应，酸性条件便于KMnO4滴定
 做对了：加入H2SO4后反应停止
 做错了：不加H2SO4导致反应继续

步骤3：滴定
 用0.1mol/L KMnO4标准溶液滴定
 至出现粉红色(30min内不消失)为终点
 为什么：KMnO4与剩余H2O2反应，粉红色表示H2O2已完全反应
 做对了：粉红色稳定30min不消失
 做错了：滴过终点导致数据偏高

八、安全红线
1. KMnO4有强氧化性，戴手套操作
2. H2SO4有强腐蚀性，戴护目镜
3. 30% H2O2有强氧化性，避免接触皮肤

九、新手避坑提示
1. KMnO4和H2O2临用前要重新标定
2. 对照用煮沸灭活的酶液，不是不加酶液
3. 滴定速度要慢，接近终点时逐滴加入
4. 粉红色要稳定30min才算终点
5. 酶液提取全程低温

十、数据处理与结果判读
计算公式：
CAT活性(mg H2O2/g FW/min) = (A - B) × 1.7 × V / (Vs × W × t)
A = 对照KMnO4滴定毫升数
B = 酶反应后KMnO4滴定毫升数
V = 提取酶液总量(mL)
Vs = 反应所用酶液量(mL)
W = 样品鲜重(g)
t = 反应时间(min)
1.7 = 1mL 0.1mol/L KMnO4相当于1.7mg H2O2

十一、常见问题排查表
问题：滴定终点不明显 → 原因：KMnO4浓度不准 → 解决：重新标定KMnO4
问题：对照和测定差异小 → 原因：酶活性低 → 解决：增加酶液量或用更嫩的组织
问题：粉红色消失 → 原因：H2O2未完全反应 → 解决：继续滴定至粉红色稳定
